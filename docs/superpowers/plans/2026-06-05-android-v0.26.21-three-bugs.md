# Android v0.26.21 — Three-bug fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Чинит три бага свежеустановленного APK: phantom-tunnel (туннель «healthy» при 0 трафика), FileProvider crash в Logs → Share, и debug-report без логов.

**Architecture:** Шесть атомарных задач — четыре фикса + диагностика + финальный bump/tag. Каждая задача независимо собирается через `tools/ship-android.sh`, инкрементирует `versionCode` и проверяется на реальном устройстве (Samsung Tab S11, `adb devices` → `R3GL207YE8P`) согласно user instructions «после каждого fix — install + проверить сценарий + logcat». Tests = device behavior (нет CI для supervisor), но добавляем 2 Rust unit-теста где это уместно.

**Tech Stack:** Rust 1.83 (tokio, socket2, tracing) для `crates/client-*`; Kotlin + Jetpack Compose для `apps/android/app/`; Android API 36, minSdk 26; `tools/ship-android.sh` (gradle + cargo + adb install).

```kotlin
                sb.appendLine("Нет активного профиля")
            }
            sb.appendLine()
            sb.appendLine("--- Конфигурация ---")
            val cfg = config.value
            sb.appendLine("DNS: ${cfg.dnsServers.joinToString()}")
            sb.appendLine("Раздельная маршрутизация: ${cfg.splitRouting}")
            if (cfg.splitRouting) sb.appendLine("Прямые страны: ${cfg.directCountries.joinToString()}")
            sb.appendLine("Per-app режим: ${cfg.perAppMode}")
            if (cfg.perAppMode != "none") sb.appendLine("Приложений выбрано: ${cfg.perAppList.size}")
            sb.appendLine()
            sb.appendLine("--- Состояние VPN ---")
            sb.appendLine("Состояние: ${VpnStateManager.state.value}")
            sb.appendLine()

            // v0.26.21: диагностический маркер. persist=N — реальный размер
            // файла LogPersister'а на диске; tail=K — сколько строк отдал
            // tailLines(2000). Если N>0 и K=0 → баг в tailLines. Если N=0 →
            // LogPersister не пишет, копаем producer.
            val logFile = java.io.File(context.filesDir, "logs/ghoststream.log")
            val persistBytes = if (logFile.exists()) logFile.length() else -1L

            com.ghoststream.vpn.service.LogPersister.flushPending()
            val lines = com.ghoststream.vpn.service.LogPersister.tailLines(context, 2000)
            sb.appendLine("--- Логи (persist=${persistBytes}B, tail=${lines.size} строк) ---")
            if (lines.isEmpty()) {
                sb.appendLine("Логи пусты — запустите туннель чтобы накопить.")
            } else {
                lines.forEach { sb.appendLine(it) }
            }

            val dir = java.io.File(context.cacheDir, "debug")
            dir.mkdirs()
            val file = java.io.File(dir, "ghoststream-debug.txt")
            file.writeText(sb.toString())
            val uri = androidx.core.content.FileProvider.getUriForFile(
                context,
                "${context.packageName}.fileprovider",
                file,
            )
            val intent = android.content.Intent(android.content.Intent.ACTION_SEND).apply {
                type = "text/plain"
                putExtra(android.content.Intent.EXTRA_STREAM, uri)
                putExtra(android.content.Intent.EXTRA_SUBJECT, "GhostStream VPN Debug Report")
                addFlags(android.content.Intent.FLAG_GRANT_READ_URI_PERMISSION)
            }
            context.startActivity(
                android.content.Intent.createChooser(intent, "Поделиться отладочной информацией")
            )
        }.onFailure { e ->
            android.util.Log.e("SettingsViewModel", "shareDebugReport failed", e)
            android.widget.Toast.makeText(
                context,
                "Не удалось собрать отчёт: ${e.message}",
                android.widget.Toast.LENGTH_LONG,
            ).show()
        }
    }
}
```

Объяснение изменений:
- Тело coroutine целиком завёрнуто в `runCatching { … }.onFailure { … }` — Exception больше не глотается `viewModelScope`.
- `onFailure` пишет `Log.e` + Toast пользователю с реальной причиной.
- Перед `--- Логи ---` секцией измеряем `logFile.length()` напрямую и кладём в маркер `persist=N B`.
- `lines.size` идёт в маркер `tail=K строк`. Видно сразу есть ли расхождение между диском и tailLines.

В `supervise.rs`, найти `let telem_task = spawn_telem_task(telemetry.clone(), status_tx.clone());` (~стр. 795) и добавить под ним:

```rust
// v0.26.21: death watcher — следит за streams_alive. Если все streams
// упали на ≥ALL_STREAMS_DEAD_TIMEOUT_SECS → шлёт dead_watcher_tx, supervisor
// делает teardown + reconnect. Если streams_up < n_streams/2 → выставляет
// health=Degraded + last_error="degraded: X/Y".
let dead_watcher = {
    let tele = telemetry.clone();
    let status_tx_w = status_tx.clone();
    let n_streams = n_streams;
    let dead_tx = dead_watcher_tx; // moved into task
    tokio::spawn(async move {
        let mut dead_streak_secs: u32 = 0;
        let half = (n_streams + 1) / 2;
        let mut last_degraded_marker: Option<usize> = None;
        loop {
            tokio::time::sleep(std::time::Duration::from_secs(1)).await;
            let alive = (0..n_streams)
                .filter(|i| tele.streams_alive[*i].load(std::sync::atomic::Ordering::Relaxed))
                .count();

            if alive == 0 {
                dead_streak_secs += 1;
                if dead_streak_secs >= crate::ALL_STREAMS_DEAD_TIMEOUT_SECS {
                    tracing::warn!(
                        category = "tunnel",
                        event = "all_streams_dead",
                        streak_secs = dead_streak_secs as u64,
                        n_streams = n_streams as u64,
                        "death watcher: 0/{} streams alive — triggering reconnect",
                        n_streams,
                    );
                    let mut f = status_tx_w.borrow().clone();
                    f.streams_up = 0;
                    f.last_error = Some(format!("all streams dead ({}s)", dead_streak_secs));
                    f.health = ghoststream_gui_ipc::TunnelHealth::Dead;
                    let _ = status_tx_w.send(f);
                    let _ = dead_tx.send(()); // signal supervisor select!
                    return;
                }
            } else {
                dead_streak_secs = 0;
            }

            if alive < half {
                // Не повторно публикуем frame при том же `alive` — снижаем
                // status churn.
                if last_degraded_marker != Some(alive) {
                    last_degraded_marker = Some(alive);
                    tracing::warn!(
                        category = "tunnel",
                        event = "degraded",
                        streams_up = alive as u64,
                        n_streams = n_streams as u64,
                        "death watcher: degraded {}/{}",
                        alive,
                        n_streams,
                    );
                    let mut f = status_tx_w.borrow().clone();
                    f.streams_up = alive as u8;
                    f.health = ghoststream_gui_ipc::TunnelHealth::Degraded;
                    f.last_error = Some(format!("degraded: {}/{} streams", alive, n_streams));
                    let _ = status_tx_w.send(f);
                }
            } else if last_degraded_marker.is_some() {
                // Streams восстановились (например, реконнект внутри drive_tunnel).
                last_degraded_marker = None;
                let mut f = status_tx_w.borrow().clone();
                f.streams_up = alive as u8;
                f.health = ghoststream_gui_ipc::TunnelHealth::Healthy;
                f.last_error = None;
                let _ = status_tx_w.send(f);
            }
        }
    })
};
```

Важно:
- `n_streams: usize` уже в scope `drive_tunnel`, аналогично `telemetry` и `status_tx`. Внутрь `tokio::spawn` они захватываются move'ом — поэтому `let tele = telemetry.clone();` и `let status_tx_w = status_tx.clone();` перед `tokio::spawn`.
- **Move `dead_watcher_tx` в task явно** — добавить выше `tokio::spawn`:
  ```rust
  let dead_watcher_tx_moved = dead_watcher_tx;
  ```
  Затем внутри task, перед `return`, вызвать:
  ```rust
  let _ = dead_watcher_tx_moved.send(());
  return;
  ```
  Это гарантирует что `dead_watcher_rx` в select! гарантированно получает сигнал (а не полагается на drop).

```rust
#[cfg(test)]
mod death_watcher_tests {
    use std::sync::atomic::{AtomicBool, Ordering};
    use std::sync::Arc;

    #[tokio::test]
    async fn all_streams_dead_triggers_teardown_within_threshold() {
        // streams_alive = [false; 4] for n_streams=4
        let alive: Vec<Arc<AtomicBool>> = (0..4).map(|_| Arc::new(AtomicBool::new(false))).collect();
        let n_streams = 4usize;
        let mut dead_streak_secs = 0u32;

        // Simulate watcher loop for 5 iterations (5 seconds).
        for _ in 0..5 {
            let count = alive.iter().filter(|a| a.load(Ordering::Relaxed)).count();
            if count == 0 {
                dead_streak_secs += 1;
                if dead_streak_secs >= crate::ALL_STREAMS_DEAD_TIMEOUT_SECS {
                    break;
                }
            } else {
                dead_streak_secs = 0;
            }
        }
        assert!(dead_streak_secs >= crate::ALL_STREAMS_DEAD_TIMEOUT_SECS);
        let _ = n_streams; // unused in this synthetic test
    }

    #[tokio::test]
    async fn partial_alive_resets_streak() {
        let alive: Vec<Arc<AtomicBool>> = (0..4).map(|_| Arc::new(AtomicBool::new(false))).collect();
        let mut dead_streak_secs = 0u32;

        // 2 dead seconds, then 1 stream comes alive, then 2 more dead.
        alive[0].store(false, Ordering::Relaxed);

        // tick 1, 2 — all dead.
        for _ in 0..2 {
            let count = alive.iter().filter(|a| a.load(Ordering::Relaxed)).count();
            if count == 0 { dead_streak_secs += 1; } else { dead_streak_secs = 0; }
        }
        assert_eq!(dead_streak_secs, 2);

        // tick 3 — one stream alive (resets streak).
        alive[0].store(true, Ordering::Relaxed);
        let count = alive.iter().filter(|a| a.load(Ordering::Relaxed)).count();
        if count == 0 { dead_streak_secs += 1; } else { dead_streak_secs = 0; }
        assert_eq!(dead_streak_secs, 0);
    }
}
```

(Тесты упрощают логику до счётчика — не используют real tokio::time::sleep, держим их быстрыми.)
